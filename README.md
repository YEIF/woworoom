#  WOWOROOM 簡易電商網站

前台: https://yeif.github.io/woworoom/

後台: https://yeif.github.io/woworoom/admin

## 功能

* 前台
    * 取得商品資料
    * 類別篩選
    * 加入購物車
    * 更改商品數量
    * 刪除單筆/全部商品
    * 送出訂單
* 後台
    * 取得訂單資料
    * 訂單詳細資料
    * 訂單狀態更改
    * 刪除單筆/所有訂單
    * 圓餅圖
